# Ramen Menu

### Description
A simple single-page React.JS mini project. 

The menu items are from <a href="https://www.ichiranusa.com/menu/#menu">ICHIRAN</a> while the images were all from Google searches.

- Multi tab
- Grid layout
- Mobile view


### Techniques
